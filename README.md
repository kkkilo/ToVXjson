# ToVXjson
转化标准格式的json文件成微信小程序所支持的对象嵌套的json文件

### 输入json格式要求
* 在ToVXjson.py文件同目录放置data.json文件（即你所要转化的文件）
* 最外面要有中括号包围;
* 中括号内的每个最外围的对象（即紧跟在中括号后面的{}包围的同层的整体对象）之间用，隔开;
* 必须是压缩后的json文件
* 压缩后的json文件里面不允许有多余的空格
